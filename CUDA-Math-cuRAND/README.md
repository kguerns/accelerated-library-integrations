# cuRAND

## Purpose & Prerequisites

cuRAND is a CUDA library for high-quality pseudorandom and quasirandom numbers. 

cuRAND is a simple bit-mixing algorithm. 

Prerequisites: Any NVIDIA GPU, nvcc and CUDA 3.2 or higher. 

## Installation & Basic Functionality
Linux: install the CUDA toolkit, including NVCC ( NVIDIA CUDA Compiler )

Brev: buy brev credits, pick a cloud GPU, run the VM from CLI.

## Relevant Use Case

Integration: Financial services customers refer to the NVIDIA Quantitative Portfolio Optimization developer example to develop applications for quantitative financial analysis. This dev example uses cuOpt, which is decision-optimization engine. cuOpt allows organizations to solve complex mathematical and logistical problems—like vehicle routing, linear programming, and job scheduling—at massive scales and near real-time speeds. cuOpt makes use of cuRAND.
